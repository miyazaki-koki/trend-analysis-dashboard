# トレンド分析ダッシュボード 2025-2026

2025〜2026年のトレンド分析と未来展望を可視化するNext.js製ダッシュボードアプリです。

## 概要

ChatGPTとの会話をもとに、SNS・AI・消費・コミュニティの構造変化を分析・可視化したWebアプリです。

### 3つの構造変化

1. **注意（Attention）の希少化** - 供給過多時代のアテンション経済の成熟
2. **信頼（Trust）の再設計** - AI生成・炎上・詐欺疑念による信頼の揺らぎ
3. **行為（Agency）の外部化** - 意思決定〜実行をAIに委譲する動き

## 技術スタック

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Recharts** (グラフ・チャート)
- **Lucide React** (アイコン)

## 機能

- 🌐 世界・日本のトレンドスコア可視化（レーダーチャート）
- 📊 技術進化の循環パターン可視化（棒グラフ）
- 🏢 長命・短命サービスの特徴比較
- 📱 レスポンシブデザイン対応

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ブラウザで http://localhost:3000 を開く
```

## 分析内容

### 世界のトレンド（2025〜2026）
- SNS：娯楽・仕事・購買の一体化（WorkTok、ソーシャルコマース）
- AI：生成AI → エージェント化とガバナンス
- 消費：便利さの飽和 → 信頼・体験・文脈へ

### 日本のトレンド
- コミュニティ：推しと界隈が社会インフラ化
- 消費：リセール・循環が購買の一部に
- 反動：デジタル成熟の裏でアナログ志向が増加

## キーメッセージ

> 「供給過多の時代に、注意と信頼と主体性をどう設計し直すか」
